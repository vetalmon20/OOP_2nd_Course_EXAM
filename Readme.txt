Варіант 32

Використовуючи засоби ООП, описати музичні гурти, що випускають пісні. Для гуртів відома назва, вподобання щодо жанрів музики, раніше випущені пісні. Для пісні відома назва, жанр, дата випуску. Для пісень розраховується популярність в заданий момент часу. Популярність залежить від кількох факторів різних типів:
- Випадковий фактор;
- Час, що пройшов з моменту випуску пісні;
- Популярність інших пісень цього гурту;
- Популярність інших пісень цього жанру;
- Відповідність жанру цієї пісні та жанрів попередніх пісень того ж гурту.
Для кожної пісні актуальні всі чи частина з цих факторів. Кожен фактор вносить свій внесок до загальної популярності пісні.
Реалізувати алгоритм для обчислення рейтингу пісень за популярністю на заданий момент – серед усіх пісень, пісень заданого жанру, пісень заданого гурту.
В інтерфейсі користувача реалізувати перегляд дерева всіх жанрів та гуртів, можливість вибрати гурт і подивитись деталі (вподобання щодо жанрів, таблицю раніше випущених пісень). Можливість сортування списку пісень за часом випуску або за популярністю (в заданий момент часу). Можливість пошуку/фільтрування списку гуртів за потраплянням до вершини рейтингу.
